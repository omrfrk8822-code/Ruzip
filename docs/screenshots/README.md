# Ekran Görüntüleri

Bu klasöre aşağıdaki ekran görüntülerini ekle:

| Dosya | Açıklama |
|-------|----------|
| `dropzone.png` | Uygulama açılış ekranı (ZIP sürükle-bırak alanı) |
| `archive_view.png` | Arşiv içeriği görünümü (dosya listesi, toolbar, statusbar) |
| `about.png` | Hakkında penceresi |

> `npm run tauri dev` ile uygulamayı çalıştır, ekran görüntüsü al ve buraya kaydet.
